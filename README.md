# iblank

自定义 Chrome 空白页（New Tab）扩展。

## 功能

- 三轨道布局：左侧搜索、中间分组、右侧预留
- 中间轨道 8 列网格，分组卡片宽度 2~8 格可调
- 分组卡片内可收藏网页地址，容量由卡片大小决定（如 4×4 = 16 个）
- 地址可独立展示，也可拖动到分组卡片内
- 自动抓取网站 favicon，支持手动上传图标
- 分组/地址数据通过 `chrome.storage.sync` 持久化

## 安装

1. 打开 Chrome 扩展管理页：`chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目目录
5. 打开新标签页即可使用

## 目录结构

```
iblank/
├── manifest.json      # 扩展清单
├── newtab.html        # 新标签页入口
├── css/
│   └── style.css      # 样式
├── js/
│   └── newtab.js      # 交互逻辑
└── icons/             # 扩展图标
```

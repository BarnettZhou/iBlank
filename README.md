# iBlank

一款简洁、可自定义的 Chrome / Edge 新标签页扩展。用模块化卡片打造你的专属起始页。

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## 特性

- **6 列栅格布局**：所有组件严格对齐在 6×N 的栅格中，自动吸附，避免重叠。
- **多种组件**：时钟、日历、便签、倒计时、天气、网页快捷方式、收藏夹。
- **编辑模式**：点击右上角「编辑」进入编辑模式，显示栅格虚线，可拖拽/调整组件大小；默认模式隐藏栅格，界面更清爽。
- **自定义壁纸**：点击 🖼️ 选择本地图片作为页面背景，右键可清除。
- **毛玻璃风格**：搜索框与组件卡片均采用半透明 + 背景模糊效果。
- **数据持久化**：组件位置与配置保存到 `chrome.storage`（扩展环境）或 `localStorage`（本地调试）。

## 组件说明

| 组件 | 默认尺寸 | 说明 |
|------|---------|------|
| 网页 | 1×1 | 显示网站 favicon，点击跳转 |
| 收藏夹 | 2×2 | 列表或图标网格布局，可添加多个书签 |
| 时钟 | 2×1 | 实时显示时间与日期 |
| 倒计时 | 2×1 | 支持具体日期或每日倒计时 |
| 日历 | 2×2 | 月视图，可切换月份 |
| 便签 | 2×2 | 多行文本，自动保存 |
| 天气 | 2×1 | 展示城市名称（演示） |

## 安装

1. 下载或克隆本仓库：
   ```bash
   git clone https://github.com/BarnettZhou/iBlank.git
   ```
2. 打开 Chrome/Edge 扩展管理页：`chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目目录
6. 打开新标签页即可使用

## 使用

- **添加组件**：点击右上角「+」选择组件类型。
- **编辑布局**：点击「编辑」按钮，拖拽移动组件，拖动右下角调整大小；组件之间不会重叠。
- **配置/删除**：右键组件可打开配置或删除菜单。
- **设置壁纸**：点击 🖼️ 选择本地图片；右键该按钮可清除壁纸。
